Java implementation of CommonMark for parsing markdown and rendering to HTML (core library)

Example:

    import org.krvlado.commonmark.node.*;
    import org.krvlado.commonmark.parser.Parser;
    import org.krvlado.commonmark.renderer.html.HtmlRenderer;

    Parser parser = Parser.builder().build();
    Node document = parser.parse("This is *Sparta*");
    HtmlRenderer renderer = HtmlRenderer.builder().escapeHtml(true).build();
    renderer.render(document);  // "<p>This is <em>Sparta</em></p>\n"

See the following packages for details:

  • {@link org.krvlado.commonmark.parser} for parsing input text to AST nodes
  • {@link org.krvlado.commonmark.node} for AST node types and visitors
  • {@link org.krvlado.commonmark.renderer.html} for HTML rendering
